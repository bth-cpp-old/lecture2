Iterativa Satser - for

  • Itererar så länge condition är true
  • Exekveringsordning:
    1. Exekvera initializer
    2. Evaluera condition
      • Om false: avsluta loop
      • Om true: gå till 3
    3. Exekvera kodblock
    4. Exekvera expression, gå till 2
